Map Integration

地图插入方式

当前默认：Leaflet + OpenStreetMap无需 Key，打开网页即可显示真实底图、标点和路线。缺点是国内 POI 细节和导航能力弱一些。
高德 JS API适合国内自驾导航。URL 后加：
?provider=amap&key=你的Key&scode=安全密钥
百度 WebGL API适合百度生态。URL 后加：
?provider=baidu&ak=你的AK

Use on phone

手机上怎么用

点左侧地点卡片，地图会定位到该地点；点地图标记可看类型和备注。
点“打开高德导航”会用当天第一个点到最后一个点生成导航链接。中间途经点仍按页面路线顺序执行。
需要真实高德底图时，把高德 Key 放在 URL 参数里，不写进仓库，避免 Key 被长期暴露。
坐标为行程规划近似标点，正式驾驶请以高德/百度实时导航和酒店确认地址为准。Leaflet/OSM 地图用于路线感知，不替代国内实时路况导航。
